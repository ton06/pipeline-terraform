🎮 APÓS O DEPLOYMENT:

Acessar o servidor:
  http://<seu-ip>:8000

SSH para debugging:
  ssh -i ~/.ssh/sua-chave.pem ubuntu@<seu-ip>

Ver logs:
  docker logs windrose-server
  docker logs -f windrose-server  # tempo real

Verificar status:
  docker ps
  terraform output

Parar o servidor (sem destruir):
  aws ec2 stop-instances --instance-ids <id>

Reiniciar:
  aws ec2 start-instances --instance-ids <id>

Destruir tudo (quando não precisar mais):
  terraform destroy -var-file="environments/dev/terraform.tfvars"
